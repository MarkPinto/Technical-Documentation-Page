Introduction

Hypertext Markup Language is the standard markup language for creating web pages and web applications. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page.

HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets. Tags such as and directly introduce content into the page. Other tags such as surround and provide information about document text and may include other tags as sub-elements.

Browsers do not display the HTML tags, but use them to interpret the content of the page. HTML can embed programs written in a scripting language such as JavaScript, which affects the behavior and content of web pages.

Your First Web Page

In order to get started with your first web page all you need to do is open up your text editor of choice and start with the most basic skeleton of an HTML document.

Start off your page with the following:

			            	
<!DOCTYPE html>
<html>
	<head>
		<title></title>
	</head>
	<body>
			            		
	</body>
</html>

Now fill in the title of the page and add content to the body such as a <p> element.

			            	
<!DOCTYPE html>
<html>
	<head>
		<title>My Web Page</title>
	</head>
	<body>
		<p>Welcome to my web page</p>		            		
	</body>
</html>

Save the above HTML document with the .html extension and when you open it up with a web browser it will result in a web page that displays the text "Welcome to my web page" as well as the title "My Web Page" on the tab that has your document open.

Adding Style

HTML is used to organize the content on the page, the content on the page can be further enhanced through styling with the use of CSS which is also known as cascading style sheets. In order to start styling your HTML, start by connecting a sheet to your document using the <link> element in the header as shown below.

			            	
<!DOCTYPE html>
<html>
	<head>
		<title>My Web Page</title>
		<link rel="stylesheet" type="text/css" href="./style.css">
	</head>
	<body>
		<p>Welcome to my web page</p>		            		
	</body>
</html>

The style element above connects a CSS document named style.css in the same directory as the HTML document. After making this link it's time to make the CSS file. This can be done using the same software mentioned in the getting started section. Let's start by making our paragraph element red. This can be done through two methods, one involves styling all <p> elements and the other involves making use of classes which let you style more selectively. Lets start by styling all <p> elements.

			            	
p
{
		color: #CCCCFF;
}

Now all <p> elements are the color periwinkle. Now let's try using classes, your CSS will not chnage much, we will call our class periwinkle.

			            	
.periwinkle
{
		color: #CCCCFF;
}

As you can see we only replaced the element that was defined which the <p> element and replaced it with a name with a period at the begining. Now in order to selectively style we must add class="periwinkle" within the opening brackets of the element we choose to stlye.

			            	
<!DOCTYPE html>
<html>
	<head>
		<title>My Web Page</title>
		<link rel="stylesheet" type="text/css" href="./style.css">
	</head>
	<body>
		<p class="periwinkle">Welcome to my web page</p>		            		
	</body>
</html>

That is the basics of CSS, there are many styling options besides color and many other methods to style elements but this will get you up and running.
